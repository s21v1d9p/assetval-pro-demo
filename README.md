## Overview

AssetVal Pro is a custom web-based application for automating the valuation of assets and goods. It calculates both **Market Value** and **Liquidation Value** using predefined formulas and comparison methodologies, with professional PDF report generation.

## Features

### ðŸ’° Valuation Calculations
- **Market Value** - Based on comparable sales, depreciation, and condition adjustments
- **Liquidation Value** - Quick sale estimate using configurable factors

### ðŸ“Š Valuation Methodology
- **Depreciation Engine** - Straight-line depreciation based on useful life
- **Condition Factors** - Excellent (95%), Good (80%), Fair (60%), Poor (40%), Salvage (20%)
- **Market Approach** - Weighted average of depreciated value (40%) + market comparable (60%)
- **Economic Adjustments** - Market demand and economic conditions modifiers

### ðŸ”„ Asset Comparison
- Side-by-side comparison of multiple assets
- Portfolio summary with totals and averages

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
